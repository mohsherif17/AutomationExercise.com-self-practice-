🧪Automation Practice – UI Test Automation Project
📖 Overview
This project is a personal training automation framework created to practice and apply core automation testing concepts.
It automates key functionalities on the Automation Practice website such as registration, login, product search, add to cart, and checkout.
The framework is designed with clean architecture, reusable components, and data-driven testing using JSON files and property configurations

⚙️ Tech Stack
•	Language: Java
•	Frameworks: Selenium, TestNG
•	Reporting: Allure Reports
•	Data Files: JSON (test-data), .properties configuration
•	Design Pattern: Page Object Model (POM)
